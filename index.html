Welcome to Jekyll Boilerplate

I’m experimenting with Jekyll and Astro. This is the Jekyll version. This is the Jekyll version, there is also an Astro version.

Welcome to Jekyll Boilerplate. A super-lightweight, accessible, SEO friendly starter project for the Jekyll site generator.

The Jekyll Boilerplate source code is available on GitHub.

Jekyll Boilerplate is built on some strong opinions:

  • Accessibility is not just important, it’s fundamental
  • Everything should still work without JavaScript
  • The democratising force in the World Wide Web is HTML files
  • Software supply chains should be minimal

Jekyll Boilerplate minimises third-party dependencies. There aren’t any plugins, jQuery isn’t in the picture, and there are no front-end frameworks.

It’s just majestic HTML, simple CSS, and some minor non-essential enhancements in plain JavaScript. Steve Fenton

Out of the box, Jekyll Boilerplate is a “top scores” high-performance website in the Pagespeed lab. Pagespeed Insights tests website performance on mobile and desktop across a number of measurements.

Test scores aren’t everything, but Jekyll Boilerplate is light and fast.
